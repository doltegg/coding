```python
import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

t=np.arange(0,5,0.2)
print("t: " + str(t))
plt.plot(t,t,'r--', t, t**2, 'bs', t,t**3, 'g^')
```
matplot 的強大畫圖功能，
先產出數值介於 0~5  的array，
並以 0. 2為間隔，
畫 出y=x, y=x², y=x³ 的圖表，
r 代表紅色、 兩個 - 代表虛線，
b 代表藍色、s 代表方塊，g 代表綠色，
^ 代表三角形。

在 jupyter 按下 tab+shift 顯示出的 plt.plot 用法。


長條圖：
```plt.bar(x,y)```
散布圖：
```plt.scatter(x,y)```
