### HTML

```HTML
<ol>
  <li>項目</li>
  <li>項目</li>
  <li>項目</li>
</ol>
<div>
  <ul>
    <li>首頁</li>
    <li>簡介</li>
    <li>作品</li>
    <li>聯絡方式</li>
  </ul>
</div>
```


### CSS

```CSS
li{
  display: inline-block;  /* 橫向 */
  border: solid 1px black;
  border-radius: 5px;  /* 圓角 */
  
  padding: 5px;  /* 字與框的距離 */
  padding-left: 20px;  /* 字與左框線的距離加 20px */
  padding-right: 20px;  /* 字與右框線的距離加 20px */
  
  font-family: 微軟正黑體;
  letter-spacing: 1px;
  
  background-color: #f7f7f7f;
}
````
