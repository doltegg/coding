### 月亮
用動畫製作有上、下弦月的月亮。




### HTML

```HTML
<div class="container">
	<div class="moon">
	</div>
</div>
```


### CSS
.container{
	margin: 100px;
}
.moon{
	width: 50px;
	height: 50px;
	border-radius: 100%;
	box-shadow: 15px -15px;
  animation: moonmove 2s infinite;
}
		
@keyframes moonmove{
    0% { box-shadow: 15px -15px; 
			   transform: translateX(-15px) translateY(15px);}
  100% { box-shadow: -30px 30px; 
			   transform: translateX(30px) translateY(-30px);}
}
````


### 結果

<div class="container">
	<div class="moon">
	</div>
</div>
